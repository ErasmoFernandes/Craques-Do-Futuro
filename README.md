# Craques do Futuro

Projeto de site para uma escola de futebol infantil.

O objetivo do projeto é apresentar a escola, mostrar sua missão, benefícios, categorias de treinamento, galeria, vídeo e formas de inscrição.

No momento, o projeto ainda está em andamento.
A página Home já foi desenvolvida e finalizada, com estrutura em HTML e estilização em CSS.

## Tecnologias utilizadas

* HTML
* CSS
* Font Awesome
* Google Fonts
